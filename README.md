### Кейс: организация полного цикла работы с данными для аналитической отчётности

**Цель**

Постороить систему аналитической отчётности, которая автоматизирует сбор и обработку данных из разнородных источников и будет предоставлять руководству ежемесячные интерактивные отчёты с ключевыми метриками.

**Этапы реализации**

**1. Определение и настройка источников данных**  
Основные данные, необходимые для аналитики хранятся в 1С:Управление торговлей и 1С:Бухгалтерия предприятия. Для выгрузки данных из них были настроены необходимые конфигурации отчетов и 
ежемесячная рассылка этих отчётов из 1С. Также данные хранятся в нескольких Excel‑файлах, которые заполняются пользователями вручную.

**2. Подготовка данных**  
Разработаны процессы в Pentaho Data Integration для:
  - загрузки данных
  - приведения их к единому формату
  - обогащения данных дополнительной информацией
  - очистки от дублей, ошибок и несоответствий

**3. Организация DWH**  
Развёрнута база на PostgreSQL для хранения подготовленных аналитических данных и настроены процессы загрузки подготовленных данных из Pentaho в DWH.

**4. Моделирование и визуализация в Power BI**  
В Power BI Desktop созданы:
  - подключения к DWH и запросы для загрузки, агрегации и преобразования данных; 
  - модели данных с необходимыми связями и иерархиями;
  - расчёты ключевых метрик с использованием DAX.
Разработаны интерактивные дашборды с визуализацией ключевых метрик с возможностью фильтрации по датам и филиалам.
Отчёты опубликованы в Power BI Service и обеспечен доступ для пользователей.

**Результат**
Результат работы:
  - интерактивные отчеты в Power BI для руководства, позволяющие анализировать ключевые метрики и принимать решения на основе наглядной аналитики
  - возможность быстрого формирования отчетов по запросу.

**Ключевые технологии:** 1С, Excel, Pentaho Data Integration, PostgreSQL, Power BI (Power Query, DAX), Power BI Service.
